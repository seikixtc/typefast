# Setting Up TypeFast on GitHub

Follow these steps to publish TypeFast to your GitHub repository.

## Step 1: Create a New Repository on GitHub

1. Go to [github.com](https://github.com) and log in
2. Click the **+** icon in the top right, then **New repository**
3. Name it: `typefast` (or your preferred name)
4. Add description: "Terminal-based adaptive typing practice with progressive key learning"
5. Choose **Public** (so others can use it)
6. Check **Add a README file** (optional, we'll replace it)
7. Choose license: **MIT License**
8. Click **Create repository**

## Step 2: Upload Files to GitHub

### Option A: Using Git (Recommended)

```bash
# Navigate to where you want to create the project
cd ~/projects  # or your preferred location

# Create project directory
mkdir typefast
cd typefast

# Copy your files here
cp ~/Downloads/typefast.py .
cp ~/Downloads/README.md .
cp ~/Downloads/QUICKSTART.md .
cp ~/Downloads/ADVANCED.md .
cp ~/Downloads/test_typefast.py .

# Initialize git repository
git init

# Add files
git add .

# Commit files
git commit -m "Initial commit - TypeFast adaptive typing practice"

# Add your GitHub repository as remote
# Replace YOUR_USERNAME with your actual GitHub username
git remote add origin https://github.com/YOUR_USERNAME/typefast.git

# Push to GitHub
git branch -M main
git push -u origin main
```

### Option B: Using GitHub Web Interface (Easier for beginners)

1. On your repository page, click **Add file** > **Upload files**
2. Drag and drop all 5 files:
   - `typefast.py`
   - `README.md`
   - `QUICKSTART.md`
   - `ADVANCED.md`
   - `test_typefast.py`
3. Add commit message: "Initial commit"
4. Click **Commit changes**

## Step 3: Update the Download URL

After uploading to GitHub, update the README.md file:

1. Open `README.md` on GitHub
2. Click the pencil icon (Edit)
3. Find the Quick Install section
4. Replace `YOUR_USERNAME` with your actual GitHub username:

```bash
curl -O https://raw.githubusercontent.com/YOUR_USERNAME/typefast/main/typefast.py
chmod +x typefast.py
```

5. Click **Commit changes**

## Step 4: Test the Download Link

Try downloading your script using the curl command to make sure it works:

```bash
curl -O https://raw.githubusercontent.com/YOUR_USERNAME/typefast/main/typefast.py
```

If it downloads successfully, you're done!

## Sharing Your Project

Once it's on GitHub, you can share it by giving people:

1. **The repository URL**: `https://github.com/YOUR_USERNAME/typefast`
2. **Quick install command**:
   ```bash
   curl -O https://raw.githubusercontent.com/YOUR_USERNAME/typefast/main/typefast.py && chmod +x typefast.py && python3 typefast.py
   ```

## Making it Even Better (Optional)

### Add Topics to Your Repo
On your GitHub repo page, click the gear icon next to "About" and add topics:
- `typing-practice`
- `touch-typing`
- `cli-app`
- `terminal`
- `python`
- `adaptive-learning`

### Create a Release
1. Go to your repo page
2. Click **Releases** on the right sidebar
3. Click **Create a new release**
4. Tag: `v1.0.0`
5. Title: `TypeFast v1.0.0 - Initial Release`
6. Description: Summary of features
7. Click **Publish release**

### Add Screenshots (Optional)
Since this is a terminal app, you could add a GIF or screenshot:
1. Run the app and take a screenshot
2. Upload to your repo in an `assets/` folder
3. Reference in README.md:
   ```markdown
   ![TypeFast Demo](assets/demo.png)
   ```
